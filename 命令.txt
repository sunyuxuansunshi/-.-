

				  			linux 命令


>常用四连
	——ls		查看目录下的文件
		-l	显示为详细信息
		-h	显示为易读单位
		-d	显示参数的本身
		-A	显示隐藏文件
		-R	递归显示
	——cd		切换到指定目录内
		-cd..	返回上一级
	——pwd		显示当前所在路径		
	——man		显示命令的所有选项
>查看时间		
	——date	查看时间信息
		-s	修改时间
>关机重启			
	——poweroff	关机	
	——reboot	重启
>网络通信
	——ifconfig	查看网卡信息
	——ping	测试网络是否接通
	——route -e	查看网关
>创建
	——touch	创建文本文档 	
	——mkdir	创建目录
>查看文件
	——cat     	适合打开小文件
		-n	显示文件行数
	——less	适合打开大文件
	——grep	打开文件的特定字符串
		-v	反查找
		-i	忽略大小写
		#通配符号为^以什么开头  $以什么结尾
	——head -x	打开文件的前X行
	——tail -x	打开文件的后X行
	
>查找文件
	——find	查找指定的文件
		-name	指定名字
		-type	指定类型
			+f 文本文件
			+d 目录
			+l 快捷方式
		-size	指定大小（k、M、G）
		-user	指定用户

		-exec	用在find命令的最后面，将find查找到的信息进行处理
			{}代表find查找到的信息
>挂载、卸载
	——mount	给光盘、U盘、网络存储、分区等提供一个可以直接访问的访问点
		umount	卸载
		mount -a 	测试分区配置的自动挂载
		showmount -e显示某个主机的共享的文件
>剪切、改名、删除、拷贝
	——rm		删除		
		-f	强制删除
		-r	递归删除（删除目录）
	——mv		剪切/改名（改名为路径不变的剪切）
	——cp		拷贝
		-r	递归拷贝（拷贝目录）
>远程管理
	——ssh		远程连接指定ip地址的主机
		-X	连接的时候可以打开图形
>软件包管理
	——rmp		单个软件包管理（容易出现依赖关系）
		-q	检测软件是否安装
		-ivh	安装软件包
		-e	卸载软件包
	——yum		软件包仓库管理（可自动解决依赖关系，方便快捷）
		+install	安装
		+remove	卸载
		-y		安装/卸载不需要询问是否确认，直接确认
		+repolist	查看软件包仓库
>重定向符号命令
	——‘>’		o将'>'左边的命令的输出信息覆盖写入到右边路径的文档内（只收集正确信息）
	——'>>'	o将'>>'左边的命令的输出信息追加写入到右边路径的文档内
	——'2>'	o将'2>'左边的命令的输出信息覆盖写入到右边路径的文档内（只收集错误信息）
	——'&>'	o将'&>'左边的命令的输出信息覆盖写入到右边路径的文档内（错误、正确信息都收集）
	——''		o
>管道符号
	——'|'		o将左边的命令的结果作为右边命令的参数

>命令行下载工具
	——wget	后面加上链接可直接下载链接内容（速度贼快，理论可以达到交换机的最快速度）

>配置网卡ip地址工具
	——nmcli	配置ip地址的工具（但是此工具识别的网卡名字和正常的不一样）
		  connection show	  		查看此工具识别的网卡名字
		  connection up '识别的网卡名'	激活配置

    	完整命令： numcli connection modify '识别的网卡名' ipv4.method manual ipv4.addresses 
		  'ip地址/子网掩码  网关' connection.autoconnect yes
>用户管理
	——id		查看用户的UID、GID、基本组、扩展组
	——useradd	创建用户
		-u	指定用户UID
		-d	指定用户的家目录路径
		-s	指定登陆的解释器程序
		-G	直接把用户加入某附加组

	——usermod	重新定义用户的各种属性
		-u	修改用户UID
		-d	修改用户的家目录路径
		-s	修改登陆的解释器程序
		-G	重置用户的附加组

	——userdel	删除用户
		-r	包括用户的家目录一并删除
	——groupadd	创建附加组
		-g	指定组GID
	——gpasswd	将用户加入/移除组
		-a	将用户加入组
		-d	将用户移除组
	——groupdel	删除附加组
	——passwd	交互式设置用户密码
		 echo '密码' | passwd --stdin 用户名    取消交互式设置密码

>归档、压缩、解包
	——gzip、bzip2、xz 	默认会删除源文件，并且一次只能压缩一个文件
		-gzip	 速度快，  压缩率低
		-bzip2 速度中等，压缩率中等
		-xz	 速度最慢，压缩率最高

	——tar		 集成备份工具,  总结： 所有的操作都有f选项 必须放在所有选项的最后
		-c	 创建归档（创建归档必须有的选项）
		-f	 指定归档文件名称（创建、释放归档必须有的，必须是在选项最后面）
		-P	 保证归档内文件的绝对路径（建议有，可以知道归档文件的路径信息）
		-z、-j、-J:调用 .gz、.bz2、.xz 格式的工具进行压缩
		-t	 查看归档中的文件清单
		-x	 释放归档
		-C	 指定释放路径 （不能连用，放在归档路径后写）
	#一般来说，归档必要的格式是 tar  -cPf  路径包名  归档1  归档2 ....... 
			解包的格式是 tar  -xf   路径包名   -C   释放路径
			查看包的文件 tar  -tf   路径包名
>重启软件、配置/设置开机自启动
	——systemctl	重启软件、配置/设置开机自启动
		restart  重起
		enable   设置开机自启动
>计划任务
	——crontab
		-e（-u 用户名）	书写计划任务（和vim一样），分时日月周 

>权限管理
	——chmod	设置文件的基本权限
			chmod  [-R]  用户+权限   文件
		-R	递归设置（可一次设置目录下的所有文件的权限）

		g+s	给予目录的权限，使目录下的所有文件、目录、新建的都具有此目录一样的权限
			附加在属组的 x 位上
 			属组的权限标识会变为 s
		
	——chown	改变文件的属主或者属组
			chown  [-R]  属主：属组   文件
		-R
			
>acl访问策略
	——setfacl 	ACL权限管理命令	
		-R	递归设置
		-m	设置用户的ACL权限 （设置之后文件的后缀会变成+）
			setfacl  -m   u:用户:权限   文件
		-x	删除指定的ACL权限
			serfacl  -x   u:用户   文件
	——getfacl		查看文件的ACL权限
			getfacl  文件
>硬盘分区管理
	——lsblk	查看本机识别的硬盘
	——blkid	查看该硬盘的详细信息（UUID、文件系统）
			blkid  硬盘路径
	——fdisk	对该硬盘进行分区
			fdisk  硬盘路径
	——mkfs.ext4、mkfs.xfs、mkfs.vfat	格式化的三种模式
	——partprobe	刷新硬盘
	——df		查看已挂载的硬盘
		-h	显示为易读的单位
